Pen+ Documentation

Pen+ is a turbowarp extension that expands the pen catagory to add more blocks that should have been there or that help accelerate the development process of pen based games

If you want video tutorials please press to go to the video tutorials playlist.

These docs will not go into every block as some are pretty self explanitory.

Common Terms

U - Horizontal position on the texture typically between 0-1, in specific cases when exceeded the image will loop back around.
V - Vertical position on the texture typically between 0-1, in specific cases when exceeded the image will loop back around.
Texture - The image on a polygon.

Pen Down

This block returns whether or not the scratch pen is down or not.
Here is what the block looks like, and what every other block looks like!

Pen Attribute

This block gets the desired attribute of the pen of the desired sprite.
There are 5 different values you can get! These are,
Pen Color
Pen Saturation
Pen Brightness
Pen Transparency
and
Pen Size
Here is what the block and what two of its variants look like.

Pen Dot

This block places a dot at the desired coordinates using the current pen color and size.

Pen Line

This block draws a line between the two desired points on the scratch stage using the current pen color and size.

Pen Square

This block draws a square at the sprites position on the scratch stage using the current pen color and size.

Pen Textured Square

This block draws a square at the sprites position on the scratch stage with an image on it! Of course using the current pen color and size.
The example below uses "costume1" as the image drawn ontop of the stamped square.

Set square attribute

This block sets an attribute of a pen based square.
These can be,
Square Width
Square Height
Rotation
U size multiplier
U position offset
V size nultiplier
V position offset
Red Tint
Green Tint
Blue Tint
and
Transparency
The tint and transparency blocks are values from 0-100

Get square attribute

Like the previous block this block gets an attribute from the pen square


Set triangle point attribute

This block sets the desired attribute of the desired point of the triangle.
The values are,
U value
V value
red tint
green tint
blue tint
transparency
and
Corner Pinch
The corner pinch value shrinks the size of that corner
The tint and transparency values are from 0-100

The values are,


Draw Solid Triangle

Draws a solid triangle between the three points using the pen color.

Draw Textured Triangle

Draws a Textured triangle between the three points with the desired texture.
